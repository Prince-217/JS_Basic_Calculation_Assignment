# JavaScript Fundamentals — Operators & Control Flow

A collection of 20 mini-programs covering core JavaScript concepts: output methods, data types, scoping, operators, and control flow. Originally written in **TypeScript** and compiled down to **JavaScript** as per course instructions.

---

## 📌 Project Overview

| Detail | Info |
|---|---|
| Topic | Operators, Data Types & Control Flow |
| Language | TypeScript → compiled to JavaScript |
| Total Programs | 20 |
| Files | `index.html`, `script.ts`, `script.js` |

This project was built as a practice assignment to strengthen fundamentals in JavaScript/TypeScript — arithmetic, assignment, logical, comparison operators, ternary operator, type conversion, and conditional logic — through small, self-contained, real-world style problems (bill calculators, salary slips, leap year checker, etc.)

---

## 🧠 What's Covered

1. **The Four Outputs** – `console.log()`, `document.write()`, `alert()`, `innerHTML`
2. **Data Type Declaration** – string, number, boolean, any
3. **ES5 vs ES6 Scope** – `var` vs `let`/`const` block scoping
4. **Const Mutation** – reassignment vs mutation of array/object contents
5. **Simple Swapping** – using a temporary variable
6. **Advanced Swapping** – without a temp variable (arithmetic trick)
7. **Calculator Basics** – addition, subtraction, multiplication, division, modulus
8. **Area of a Circle** – using a constant PI
9. **Simple Interest Calculator** – `(P × R × N) / 100`
10. **Celsius → Fahrenheit Converter**
11. **Restaurant Bill Generator** – subtotal + 5% GST
12. **Discount Calculator** – 15% discount on product price
13–14. **Employee Salary Slip** – HRA, DA, Provident Fund, Gross & Net Salary
15. **Days → Years/Months/Days Converter**
16. **String → Number Conversion** – conversion vs concatenation
17. **Even or Odd Checker** – using modulus operator
18. **Maximum of Two Numbers** – using the ternary operator
19. **Student Grade Logic** – pass/fail using logical `&&`
20. **Leap Year Checker** – using logical operators

## ▶️ How to Run

1. Clone the repository
   ```bash
   git clone <your-repo-url>
   ```
2. Open `index.html` directly in your browser
   *(or use the VS Code "Live Server" extension for auto-reload)*
3. Open the browser **Console** (`F12` → Console tab) to view `console.log()` outputs
4. Watch for `alert()` popups and check the page body for `document.write()` / `innerHTML` outputs

## ✍️ Key Learnings

- Difference between `var` (function-scoped) and `let`/`const` (block-scoped)
- Why `const` prevents *reassignment* but not *mutation* of arrays/objects
- Practical use of arithmetic swap logic without a temporary variable
- Real-world use cases for operators: billing, salary slips, interest calculators
- Type conversion (`Number()`) vs string concatenation pitfalls
- Writing TypeScript with explicit types and compiling to clean, readable JavaScript
